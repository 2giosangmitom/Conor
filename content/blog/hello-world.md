# 🌐 NST Crystal – Personal Website

This repository contains the source code for my personal website and portfolio.

The website is used to showcase my projects, write blog posts, and share knowledge about programming and computer science.

## 🚀 Live Website

Visit the website here:

**https://nstcrystal.is-a.dev**

## 🖼 Preview

![Preview](/images/blog/Preview_Markdown.png)

## ⚙️ Tech Stack

- **React**
- **Vite**
- **TypeScript**
- **Markdown** (for blog posts)
- **Vercel** for deployment

## 📂 Project Structure

```text
src/
 ├─ components/   # Reusable UI components
 ├─ pages/        # Main pages (Home, Blog, About...)
 ├─ content/      # Markdown blog posts
 ├─ hooks/        # Custom React hooks
 └─ utils/        # Helper functions
```

## 🛠 Development

Clone the repository and run locally:

```bash
git clone https://github.com/nstcrystal/nstcrystal.github.io.git
cd nstcrystal.github.io
npm install
npm run dev
```

Then open:

```
http://localhost:5173
```

## 📦 Deployment

This project is deployed automatically using **Vercel**.

Every push to the `main` branch triggers a new deployment.
