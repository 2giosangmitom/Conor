# NgheGo

NgheGo is a modern English listening practice platform built with Nuxt 3.

The platform helps users improve:

- listening skills
- vocabulary
- accent recognition
- typing reflexes

using real YouTube videos.

---

# Main Features

- YouTube listening practice
- Dictation typing
- Accuracy scoring
- Progress tracking
- Streak system
- Accent-focused learning

---

# Tech Stack

- Nuxt 3
- Vue 3
- Nuxt UI
- TailwindCSS
- Lucide Icons

---

# Project Goal

The project focuses on:

- premium SaaS UI
- clean architecture
- reusable components
- responsive UX
- calm learning atmosphere
